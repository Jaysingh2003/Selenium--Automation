# Selenium Automation

A Selenium WebDriver test automation framework built with Java, Maven, and TestNG, following the Page Object Model (POM) design pattern.

## What it does

- Opens Google in Chrome
- Searches for `Selenium WebDriver`
- Waits for the results page using explicit waits (`WebDriverWait`)
- Verifies the page title contains `Selenium`

## Tech stack

- Java
- Selenium WebDriver 4.45
- TestNG
- Maven
- WebDriverManager (automatic ChromeDriver management)

## Requirements

- Java 17+ (tested on Java 25)
- Maven
- Google Chrome
